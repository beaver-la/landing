# NextJS

Para levantar el proyecto:
1. npm install -> para instalar todas las dependencias
2. npm run dev o next dev -> para correr el servidor

# Deploy
El deploy esta armado con Vercel, que se lleva muy bien con Next
La version gratuita de Vercel no permite repos de organizaciones, asi que tengo un clone de este repo en mi perfil
Tengo un remote en mi escritorio que se llama "deploy" y que apunta al clone del repo
Asi que para deployar hay que pushear a ese remote: git push deploy main (o git push deploy prod)
